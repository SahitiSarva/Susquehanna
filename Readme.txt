
This repository contains the code that supports the analysis performed and reported for the study titled 
'Operationalizing fariness and stability in transboundary water resource allocations' which is available here https://repository.tudelft.nl/islandora/object/uuid:683a47b3-89fd-4ebf-8ba6-1a2fe6c15b65\
This study uses the Susquehanna river basin model found in the analysis performed by Salazar et. al. (2016)

The repository contains the model which is converted from C++ to python and the notebooks that are used to identify fairness and stability of policies.
